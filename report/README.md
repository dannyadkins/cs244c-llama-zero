# Reports

This directory holds research notes, experiment summaries, and writeups derived from completed runs.

## Contents

- `bandwidth_tradeoff_report.md`: consolidated summary of fixed-workload and fit-to-memory evidence
- `stage2_stage3_low_bandwidth_findings.md`: diagnosis of low-bandwidth Stage 2 vs Stage 3 behavior
- `stage2_stage3_low_bandwidth_trace_report.md`: trace-level communication analysis for the corrected low-bandwidth rerun
- `stage3_memory_fit_findings.md`: notes from larger-scale Stage 3 fit-to-memory experiments
- `textbook_zero_bandwidth_report.md`: results after bringing the implementations closer to the textbook ZeRO formulation

The markdown in this directory is written to be portable across machines. Host-specific addresses and filesystem roots have been replaced with reusable placeholders so the reports remain shareable.
